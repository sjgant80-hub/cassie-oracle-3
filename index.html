§ I · what it isThe brain behind the solver.

CASSIE runs in your browser hunting BTC puzzle 135. As it walks, it broadcasts walker DP positions and 7-prime ring telemetry on BroadcastChannel('fall-signal').

Oracle 3 sits next to CASSIE as a Node CLI. You feed it the state — exported from CASSIE or POSTed via HTTP — and it:

  • Spawns 7 prime-domain specialist agents in parallel (L1) · one per spine prime [2, 3, 5, 7, 11, 13, 17]
  • Each specialist runs a 7-pass v19 recursive ritual (0d → -12d, n+1, 7 times) inside its own prompt
  • Spawns a chord-resolver (L2) only when ≥ 2 specialists co-resonate above κ-threshold
  • An L0 orchestrator integrates everything → concrete bias recommendations for CASSIE
  • Returns advice as JSON: priority zone weight changes, new zone candidates, plain-English summary

§ V · setupThree minutes from zero.

1 · install Claude Code (skip if you have it)

npm i -g @anthropic-ai/claude-code
claude               # OAuth · /quit when done

2 · clone, install

git clone https://github.com/sjgant80-hub/cassie-oracle-3
cd cassie-oracle-3
npm install

3 · run

# CLI mode · read state, print advice
node oracle3.mjs ./examples/sample-state.json

# write advice to file
node oracle3.mjs ./examples/sample-state.json ./advice.json

# server mode · CASSIE posts state, gets advice back
node oracle3.mjs --serve --port 7777
